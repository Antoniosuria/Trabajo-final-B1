¿Y un libro para aprender Javascript en castellano?

En español recomendamos el libro de nuestro profesor, Juanma Garrido. Conocerás las bases de javascript como lenguaje de programacion (variables, operadores, condicionales, bucles, tipos datos, etc...) y conceptos un poco mas avanzados pero fundamentales para entender el Javascript moderno. Puedes adquirirlo aquí: Apuntes de Javascript. También puede interesarte la introducción a Javascript de Javier Eguiluz en LibrosWeb.

Eloquent JavaScript

El primero, sin lugar a dudas es Eloquent JavaScript: A Modern Introduction to Programming, de Marijn Haverbeke. Este no es realmente un libro de JavaScript. Como sugiere el título, es una introducción a la programación usando JavaScript. Muy útil para aquellos que quieren aprender a programar desde cero. Puedes leer (y usar) la tercera edición de forma gratuita a través Eloquent Javascript. Puedes encontrar una versión en español aquí.

Foundation HTML5 Animation with JavaScript

En este libro aprenderás a utilizar una animación increíble y un código basado en la física pura. En muy poco tiempo tendrás un gran entendimiento de los conceptos de tas de animación con guión y tendrás una gran capacidad de creas todo tipo de animaciones y juegos. Lo que aprenderás en este libro: • Todo el código JavaScript y HTML5 (incluyendo matemáticas y funciones de trigonometría) tendrás que comenzar a animar con el código • Principios básicos de movimiento como velocidad, aceleración, fricción y los rebotes Cómo manejar las interacciones del usuario con el teclado, el ratón y pantalla táctil • Las avanzadas técnicas de movimiento como resortes, coordinar la conservación de la rotación de la fuerza, y la cinemática directa e inversa • Todos los conceptos básicos de 3D que usted necesita para 3D en HTML5 (sin WebGL)-desde la perspectiva simple de sólidos 3D completo, con la cara posterior sacrificio y la iluminación dinámica Este libro es para este libro es un recurso fantástico para todos los desarrolladores web que trabajan en HTML5 o Flash para crear compatibilidad con los estándares juegos, aplicaciones y animaciones que funciona en todos los navegadores modernos y la mayoría de dispositivos móviles, incluidos iPhones, iPads y dispositivos Android. Libro disponible en Amazon.

JavaScript: The Definitive Guide (Definitive Guides)

Since 1996, JavaScript: The Definitive Guide has been the bible for JavaScript programmers—a programmer's guide and comprehensive reference to the core language and to the client-side JavaScript APIs defined by web browsers. The 6th edition covers HTML5 and ECMAScript 5. Many chapters have been completely rewritten to bring them in line with today's best web development practices. New chapters in this edition document jQuery and server side JavaScript. It's recommended for experienced programmers who want to learn the programming language of the Web, and for current JavaScript programmers who want to master it.
